# Telegram бот: номер -> mp3

Бот проверяет подписку на Telegram-канал, затем принимает номер и отправляет связанный с ним mp3.

## Как работает

1. Пользователь пишет `/start`
2. Выбирает язык
3. Подписывается на канал
4. Нажимает `Проверить подписку`
5. Отправляет номер
6. Бот находит запись в `data/tracks.json`
7. Бот отправляет mp3
8. Если указана `buyUrl`, под mp3 появляется кнопка `Купить`

## Где хранить mp3

Есть 2 варианта:

1. Локально в папке `media`
2. В облаке по прямой ссылке

Бот поддерживает оба варианта.

## Формат `data/tracks.json`

Вариант с облаком:

```json
{
  "101": {
    "url": "https://site.com/audio/beat-101.mp3",
    "caption": "Ваш бит для номера 101",
    "buyUrl": "https://site.com/buy/101",
    "buyText": "Купить"
  }
}
```

Вариант с локальным файлом:

```json
{
  "102": {
    "file": "beat-102.mp3",
    "caption": "Ваш бит для номера 102",
    "buyUrl": "https://site.com/buy/102",
    "buyText": "Купить"
  }
}
```

Пояснение:

- `url` — прямая ссылка на mp3 в облаке
- `file` — имя файла из папки `media`
- `caption` — подпись под аудио
- `buyUrl` — ссылка для кнопки покупки
- `buyText` — текст кнопки

Нужно указывать либо `url`, либо `file`.

## Запуск

1. Установите `Node.js 18+`
2. Создайте `.env` по образцу из `.env.example`
3. Заполните `data/tracks.json`
4. Если используете локальные файлы, положите их в `media`
5. Запустите:

```powershell
node bot.js
```

## Пример `.env`

```env
BOT_TOKEN=123456:ABCDEF_your_bot_token
```
